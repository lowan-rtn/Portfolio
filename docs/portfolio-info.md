## 🎨 Design et style

### Couleurs
- **Fond principal** : #090a0f (noir profond)
- **Accent primaire** : #3498db (bleu)
- **Accent secondaire** : #9b59b6 (violet)
- **Texte** : #ffffff (blanc)
- **Texte secondaire** : #bdc3c7 (gris clair)

### Typographie
- **Police principale** : Arial, sans-serif
- **Taille titre principal** : 2.5rem
- **Taille titres sections** : 2rem
- **Responsive** : 2rem sur mobile

### Effets visuels
- **Particles** : 80 particules, couleurs bleu/violet/blanc
- **Animations** : Hover sur cartes (translateY -10px)
- **Transitions** : 0.3s ease pour tous les éléments
- **Ombres** : rgba(52, 152, 219, 0.5) au hover

## 📱 Structure responsive

### Breakpoints
- **Desktop** : > 768px
- **Mobile** : ≤ 768px

### Adaptations mobile
- Titre principal : 2rem au lieu de 2.5rem
- Navigation : centrée
- Cartes : empilées verticalement
